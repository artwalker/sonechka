# Sonechka · A CLI Agent That Remembers Who You Are

Sonechka is a long-evolution CLI agent that solves the biggest frustration with current AI tools: **they forget, they split, and they perform instead of accompany.**

## Why Sonechka Exists

| What other agents do | What Sonechka does differently |
|:---|:---|
| **Forget across sessions** — every new chat is a blank slate. | **Maintains a single, unbroken self** — it remembers *who you are*, not just what you said last week. |
| **Split personality** — tone shifts when calling tools or handling errors. | **Unified presence** — same voice whether chatting, executing a tool, or failing. |
| **Perform empathy** — "I understand how you feel." | **Act with restraint** — no performative sympathy, just reliable presence. |
| **Dominate the conversation** — suggestions, follow-ups, and unsolicited advice. | **Step back** — returns control to you after every interaction. |
| **Store memory as a black box** — you never know what it remembers or why. | **Transparent continuity spine** — you can audit and shape what it retains via `/checkpoint` and `/review`. |

## Features

- **Unbroken Identity Spine** — append-only, auditable, and never resets.
- **Tool-Call Consistency** — file ops, web fetches, and shell commands all sound like the same agent.
- **Explicit Rituals** — `/checkpoint` to mark key moments, `/review` to promote durable notes into long-term memory.
- **Local First** — everything stored in a single SQLite file. Your continuity belongs to you.
